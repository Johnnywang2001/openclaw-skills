# WiFi Scanner

Scan and analyze nearby WiFi networks, check signal strength, and diagnose connectivity issues. Use when asked to find available WiFi networks, check signal quality, troubleshoot WiFi, compare network speeds, find the best channel, or analyze wireless interference. Supports macOS and Linux.

## Installation

```bash
clawhub install wifi-scanner
```

## Usage

```bash
# Scan available WiFi networks
python3 scripts/wifi_scanner.py scan
# Show current connection info
python3 scripts/wifi_scanner.py status
```

## Requirements

- Python 3.7+
- macOS
